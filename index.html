Basic Structure

Every HTML document starts with a basic structure:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Global Attributes

Global attributes can be used on any HTML element:

  • class: Defines one or more class names.
  • id: Defines a unique ID.
  • style: Defines inline CSS.
  • title: Defines extra information (tooltip).
  • data-*: Used to store custom data.
  • contenteditable: Makes an element editable.
  • draggable: Makes an element draggable.
  • hidden: Hides an element.

Metadata

Metadata provides information about the HTML document:


<meta charset="UTF-8">
<meta name="description" content="A page about HTML">
<meta name="keywords" content="HTML, Web Development">
<meta name="author" content="Your Name">
<meta http-equiv="refresh" content="5">
<base href="https://example.com/">
<link rel="preload" href="image.jpg" as="image">
